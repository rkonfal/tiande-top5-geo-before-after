# GEO before/after archive

Statická GitHub Pages microsite s archivem nasazených GEO přepisů:

- původní nasazená TOP5
- finální produkční vrstva dalších 20 produktů
- silnější decision-page upgrade pro všech 25 produktů
- nová další vlna 25 produktů už nasazená po ručních passech ve směru co nejblíž 10/10
- archivní before/after pohled na title, meta, H1 a `product-description`
- blokové vysvětlení, co se měnilo

Výběr je dělaný podle YTD prodaných kusů z lokálních reportovacích dat.

Hotové části jsou defaultně sbalené a slouží jako referenční archiv toho,
co bylo skutečně nasazené do produkce.
